# ChordCraft Application - Piano Soundbank

The audio sandbank used for this application is a set of samples derived from the **Accurate-Salamander Project**, a remastering of the Salamander Grand soundbank authored by Chisato Yamauchi of the Misato Astronomical Observatory.
